ADD  Nachos, OS_Features

Tools: C++, Make

This os-build project is an extension of Nachos, a UNIX-like OS created for educational purposes, which I did in OS Lab at BUET. The vanilla Nachos is extented by a) Consumer-producer program, b) Basic multiprogramming scopes through System Calls[Exec,Exit,Read,Write] implementation and c) Virtual Memory Manager supported by Demand Paging and Page Replacement procedures: without Swapfile [ Random Replacement and LRU Replacement], with SwapFile.

DDD: Drunk Driver Detector

Platform: Node ESP8266, MQ-5, DC Motor, LCD

This project belongs to detect any alcoholic driver for avoiding unnecessary accidents. To do this we built a system using MQ-5 gas sensor which can detect alcohol gas and glow a green signal at the back of its led. We are assuming a motor as a vehicle. Now if it sense any alcohol gas then this will be displayed on the LCD display and system will block the motor. That means if any driver gets alcoholic, this system will detect that.

Amazon Echo : Home automation

And in this project, we had implemented voice controlled home automation using Amazon Alexa on Raspberry Pi 3 and ESP8266 node module.

  • Remote voice controlled system is available in the system.
